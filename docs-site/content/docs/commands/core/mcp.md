# mcp

MCP server for AI assistants

## Usage

```bash
portunix mcp [options] [arguments]
```

## Full Help

```
ptx-mcp is a helper binary for Portunix that handles all MCP (Model Context Protocol) server operations.
It provides MCP server functionality for AI assistant integration.

This binary is typically invoked by the main portunix dispatcher and should not be used directly.

Usage:
  ptx-mcp [flags]

Flags:
  -h, --help      help for ptx-mcp
  -v, --version   version for ptx-mcp

```
