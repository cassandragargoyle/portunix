# update

Update Portunix to the latest version

## Usage

```bash
portunix update [options] [arguments]
```

## Full Help

```
Usage:
  portunix update [flags]

Flags:
      --check   Only check for updates without installing
      --force   Force update even if on latest version
  -h, --help    help for update

Global Flags:
      --help-ai       Show machine-readable help in JSON format
      --help-expert   Show extended help with all options and examples

```
